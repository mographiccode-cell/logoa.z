# logoa.z - صفحة هبوط احترافية

## نبذة عن المشروع
هذه صفحة هبوط احترافية مبنية لحساب @logoa.z على إنستجرام، متخصص في التصميم الجرافيك والتسويق الرقمي. الصفحة مبنية باستخدام Tailwind CSS مع تصميم حديث ومتجاوب.

## الميزات الرئيسية
- تصميم RTL كامل مناسب للغة العربية
- متجاوب 100% (موبايل، تابلت، ديسكتوب)
- تأثيرات CSS احترافية
- أداء عالي وسرعة تحميل
- تحسينات SEO أساسية
- دعم وضع Light/Dark
- تأثيرات حركية ناعمة

## التقنيات المستخدمة
- HTML5
- Tailwind CSS ( عبر CDN )
- JavaScript ( vanilla )
- خطوط Google ( Cairo, Tajawal )

## التشغيل المحلي
1. قم بتنزيل المشروع
2. افتح ملف `landing-page.html` في المتصفح
3. ستعمل الصفحة مباشرة بدون الحاجة لخادم

## التحديث لاحقاً
1. عدّل الملفات حسب الحاجة
2. ادفع التغييرات إلى GitHub
3. سيتم النشر تلقائياً عبر GitHub Pages
